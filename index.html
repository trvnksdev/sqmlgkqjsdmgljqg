Quiz Chinois — Colle les tableaux (caractères, pinyin, traduction)

1) Importer la liste

Colle une ligne par item avec séparateur \ (espaces autour acceptés) :

汉字 \ pinyin \ traduction française

Exemples :

你好 \ nǐ hǎo \ bonjour
中国 \ zhōngguó \ Chine
Conseils rapides
  • Tu peux coller directement toute la liste des tableaux 1 & 2 + les 27 mots.
  • Format minimal possible (si pas de pinyin) : 汉字 \ \ traduction.
  • Les distracteurs (faux choix) sont générés automatiquement à partir des autres traductions de la liste.
  • Tout se passe en local (aucune donnée n'est envoyée). Le progrès est mémorisé dans le navigateur.
